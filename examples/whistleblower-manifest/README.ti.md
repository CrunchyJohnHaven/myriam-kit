   ሕታም ኔሮም።

## መዓስ እዚ ኪት ክትጥቀመሉ

ቅድሚ ናብ ጋዜጠኛ፣ ተቆጻጻሪ ወይ ጠበቓ ጭብጢ ምትሕልላፍ ማኒፈስት
ክትሓትም ክትሓስብ ኣለካ፣ እዚ ዝስዕብ ኩሉ እንተኾይኑ፥

- ብዲጂታል መልክዕ ጭብጢ ኣለካ (PDFs፣ ኢመይል፣ ምስልታት፣ ወዘተ)
  ብናይ ባይት ቅንዕና ክዕቀብ ትደሊ።
- ኣማላዲታት ነቲ ጭብጢ ክቕይሩ፣ ክከራኸሩ ወይ ክቐብሩ ይኽእሉ
  ኢልካ ትጽበ።
- ክሳብ ምትሕልላፍ መሰረታውያን ሰነዳት ኣብ ኢድካ ክተንብር ትኽእል።
- ናይ ሰነዳት ትሕዝቶ ብዘይምግላጽ ማኒፈስት ብህዝቢ ክተሕትም ትኽእል
  (ሃሽ ጥራይ ህዝባዊ እዮም)።
- ናይ ኣፍላጢ ኾይንካ ብህዝቢ ክትፍለጥ ምኽኣል ኣካዊናካ ኣለካ (ናይ
  ሕታም-ቀንዲ ቦታ)። እንተ ብስም ድኣ ኣይሎይ፣ ኣብ ኩሉ ተዛመድቲ
  ሕታማት ቅሉዕ ስም ብዝቐጸለ ኣጠቓቕማ።
- ብዛዕባ ናይ ኣፍላጢ-ምክልኻል ሕጊ ኣብ ዘመናካ ምስ ጠበቓ ተወኪስካ።

## መዓስ እዚ ኪት ዘይትጥቀመሉ

እዚ ኪት ኣይትጠቀመሉ እንተ፥

  ህዝባዊ-ሃሽ ክግበር ዘይብሉ ውልቃዊ ሓበሬታ ሓዙ። ናይ ቀንዲ
  ኣይኮነን ናይ ዝተሓዝአ ቅዳሕ ሃሽ ግበር።
- ናይ ማኒፈስት ምሕታም ባዕሉ ንኣፍላጢ ዘይተሰማማዓሉ ብዝኾነ መንገዲ
  ዝፈልዮ።
- ክሳብ ሕጂ ምስ ጠበቓ ኣይተወከስካን። ፕሮቶኮል መሳርሒ እዩ፤
  ናይ ኣፍላጢ-ምክልኻል ሕጊ ናይ ዘመን-ፍሉይ እዩን ካብ ፕሮቶኮል ወጻኢ ይነብር።

  እዩ፤ ሓደስቲ ዝተሓትሙ ምትዕርራያት ጥራይ ሓቀኛ መንገዲ ምክልባት እዮም።
- ኣብ ትሕቲ ጸቕጢ ወይ ናይ ግዜ ጸቕጢ ትገብሮ። ፕሮቶኮል ንግዱድ

## ተቐበልቲ ዝፈልጥዎ

ምስቲ ዝተሓትመ ማኒፈስት ነዚ README ናብ ተቐበልቲ ላኣኾ። እቲ ማኒፈስት
ባዕሉ ናይ ተቐባሊ ፕሮቶኮል (FACT 06) ሓዘ፣ ግን README
ናይ ተቐባሊ ሚና የስፍሕ፥

- **ጋዜጠኛታት**፥ መጀመርያ ናይ ማኒፈስት ናይ ባዕሉ ቅንዕና ኣረጋግጽ።
  ድሕሪኡ ብፈለለ ድሕነታዊ ቻነል ሰነዳት ሕተት። ቅድሚ ምሕታም
  ናይ ነፍሲ-ወከፍ ሰነድ SHA-256 ምስ ማኒፈስት ዝሰማማዕ ምዃኑ ኣረጋግጽ።
  ብዘይ ናይ ኣፍላጢ ስምምዕ ሰነዳት ኣይተሕትም፤
  ናይ ዝተሓትመ ማኒፈስት ህልውናን ናይ ምርግጋጻኻ
  መደምደምታን ኣሕትም።
- **ተቆጻጸርቲ**፥ ሓደ ዓይነት መርኣዪ ስርዓት። ዝተሓትመ ማኒፈስት ምስ
  ዝሰማማዑ ሰነዳት ካብ ናይ ምስጢራዊ ምእካብ ወጻኢ ዝቐውም
  ብሉጽ ናይ ጭብጢ መንገዲ እዩ።
- **ጠበቓታት**፥ ዝተሓትመ ማኒፈስት ኣብ ዝተባህለ ግዜ ናይ ሰነዳት
  ህልውና ናይ ፎረንሲክ መሊ ይህብ። ናይ ዝኾነ ፍሉይ ዘመን ናይ ጭብጢ
  ሕግታት ምምልኡ ኣይምልኡን ንስኻ ክትውስን ኣለካ።

## እዚ ኪት ዘይኮነ ነገራት

- ናይ ሕጋዊ ምኽሪ ምትካእ ኣይኮነን።
- ሰብ ናይ ምግላጽካ ቃል ክስዕብ ቃል ኣይኮነን።
- ካብ ሕነ ምፍዳይ ድሕነት ኣይኮነን።
- ናይ ሰነዳት ትሕዝቶ ምርግጋጽ ኣይኮነን (ናይ ሕታም ግዜ ባይታት ጥራይ)።
- ምስ ዝኾነ ሰብ ብዘይ ስም ናይ ምርኻብ መንገዲ ኣይኮነን — ሕታም
  ህዝባዊ እዩ፤ ናይ ሕታም-ቀንዲ ቦታ ስም ሓቐኛ እንተኾይኑ፣ ተፈሊጥካ ኣለኻ።

## መዕጽው

ዝተሓትመ ማኒፈስት ንእሽቶ ነገር እዩ። ሓደ ናይ ባይት ጅረት ናይ ዓቢ
ስብስብ ናይ ባይት ጅረታት ኣብ ፍሉይ ግዜ ኣብ ፍሉይ መልክዕ ኔሩ ምዃኑ ይምስክር።
እታ ንእሽቶ ሓይሉ እያ፥ ሓደ ነገር ጥራይ ይገብር፣ ኣብ ናይ Bitcoin
ብሎክቼን ህይወት ሙሉእ ነቲ ሓደ ነገር ይገብር።

እዚ ናይ ኣፍላጢ ምዃን ስለ ትሓስብ እዚ እንተ ኣንቢብካ፣ በጃኻ፥
መጀመርያ ምስ ጠበቓ ተዘራረብ። ካብቶም ኣብ ላዕሊ ዘለዉ ናይ ስራሕ
ድሕነት መሪሕታት ተጠቐም። ድሕሪኡ፣ ኩሎም እቶም ላዕለዎት
ጉዳያት ኣብ ቦታኦም እንተኾይኖም ጥራይ፣ ነዚ ኪት ናይ ምግላጽካ ክሪፕቶግራፊ
ድጋፍ ኮይኑ ሕሰብ። ክሪፕቶ እቲ ቐሊል ክፋል እዩ። ኩሉ ካብ ክሪፕቶ ዝላዕለ
እቲ ሓቀኛ ስግኣት ዝነብር እዩ።
