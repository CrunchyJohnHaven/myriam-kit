# የጥቆማ አቅራቢ ማኒፌስት ማሸጊያ ኪት

ለጥቆማ አቅራቢዎች የተዘጋጀ ዝግጁ-አገልግሎት myriam-kit ምሳሌ ሲሆን፣ የተወሰነውን የውድቀት ዘይቤ ይቋቋማል — እርሱም **በጥቆማ አቅራቢውና በሕዝብ መካከል ባሉ መካከለኛ አካላት ማስረጃው ሲቀበር፣ ሲስተካከል ወይም ሲከራከር**። ይህ ኪት የሚከላከልባቸው ቀኖናዊ ታሪካዊ ጉዳዮች:

- Theranos (የላብራቶሪ-ምርመራ ጉዳዮች ለዓመታት ታፍነዋል)

- T9: ሦስተኛ ወገኖች ለጋዜጠኞች የተቀየሩ ቅጂዎችን ማቅረብ።

- በጥቆማ አቅራቢው ላይ ለመመለስ የሚደረግ ጫና (T4፣ በከፊል በFACT 07 የተቀነሰ ሲሆን ይህም ጸጥተኛ መመለስን ሕዝባዊ ለመለየት የሚቻል ያደርጋል)።
- በራሱ ይፋ ለማድረግ ድርጊት በጥቆማ አቅራቢው ላይ ሕጋዊ እርምጃ (T5)።
- የጥቆማ አቅራቢውን ባሕርይ ማጥፋት (T8)።
- መካከለኛ አካላት ታሪኩን በጸጥታ መተው (T10)።

ይህን ኪት ለመጠቀም እያሰበ ያለ የጥቆማ አቅራቢ ከሆኑ፣ **መጀመሪያ ይህን ሙሉ README ያንብቡ**። ፕሮቶኮሉ መሳሪያ ነው። ፕሮቶኮሉ ለጠበቃ፣ ለጋዜጠኛ፣ ለተቆጣጣሪ፣ ወይም ለጥንቃቄ የሥራ ደህንነት ምትክ አይደለም።

## ይህ ኪት ምን ያመነጫል

በምስጢራዊ ምስጠራ የተሸገ የማስረጃ *ማኒፌስት* (ማስረጃው ራሱ አይደለም) ሲሆን፣ የሚከተለውን ያረጋግጣል:

1. የእያንዳንዱ የማስረጃ ፋይል ልዩ ባይቶች በማሸግ ቅጽበት ላይ መኖራቸውን።
2. ማሸግ በተወሰነ ጊዜ ላይ ወይም በፊቱ መከናወኑን (በBitcoin OTS ማረጋገጫ በኩል)።
3. የመነሻ ምንጭ (provenance)፣ ክስ፣ የስጋት ሞዴል፣ የተቀባይ ፕሮቶኮል፣ እና ጸጥተኛ-አለመመለስ ቁርጠኝነት ሁሉም በአንድ ጊዜ መታወጃቸውን።

መሠረታዊ ሰነዶቹ ለተቀባይ እስከሚላኩ ድረስ በእጅዎ ይቆያሉ። ተቀባዮች የተቀበሉት ባይቶች ከእርስዎ ካሸጉት ጋር ተመሳሳይ መሆናቸውን ያረጋግጣሉ።

## ይህን ኪት መቼ መጠቀም

ለጋዜጠኛ፣ ለተቆጣጣሪ፣ ወይም ለጠበቃ ማስረጃ ከማስተላለፍ **በፊት** ማኒፌስት ማሸግን ግምት ውስጥ ማስገባት አለብዎት፣ ከሚከተሉት ሁሉም እውነት ከሆኑ:

- በዲጂታል መልክ ማስረጃ (PDF፣ ኢሜል፣ ምስሎች፣ ወዘተ) አሎት፣ እና በባይት-ለ-ባይት ትክክለኛነት ለመጠበቅ ይፈልጋሉ።
- መካከለኛ አካላት ማስረጃውን ሊቀይሩ፣ ሊከራከሩ፣ ወይም ሊቀብሩ እንደሚችሉ ይጠብቃሉ።
- መሠረታዊ ሰነዶችን እስከ ማስተላለፊያ ድረስ በእጅዎ ማቆየት ይችላሉ።
- የሰነዶቹን ይዘት ሳይገልጹ ማኒፌስቱን በሕዝብ **ማተም ይችላሉ** (ሐሽ ብቻ ሕዝባዊ ናቸው)።
- እንደ ጥቆማ አቅራቢ ሕዝባዊ መታወቅ ሊከሰት የሚችለውን አደጋ መዝነዋል (signing-principal መስክ)። ስም ያለመግለጽ ከሆነ፣ በሁሉም ተዛማጅ ህትመቶች ላይ የተወሰነ የውሸት ስም (pseudonym) በተከታታይ ይጠቀሙ።
- በስልጣን ክልልዎ ውስጥ ስለሚገኝ የጥቆማ-አቅራቢ-ጥበቃ ሕግ ጠበቃ አማክረዋል።

## ይህን ኪት መቼ መጠቀም የለብዎትም

ይህን ኪት አይጠቀሙ ከሆነ:

- ሰነዶቹ ራሳቸው በሕዝብ-ሐሽ-መደረግ የሌለበት ስለ ሦስተኛ ወገኖች (ታካሚዎች፣ ደንበኞች፣ ሠራተኞች) የግል መረጃ ይዘዋል። የተደበቀ ስሪት ሐሽ ያድርጉ፣ ኦሪጅናሉን አይደለም።
- ማኒፌስቱን የማተም ድርጊቱ ራሱ የጥቆማ አቅራቢውን በማይፈቅዱት መንገድ ይለያል።
- ጠበቃ ገና አላማከሩም። ፕሮቶኮሉ መሳሪያ ነው፣ የጥቆማ-አቅራቢ-ጥበቃ ሕግ የስልጣን-ክልል-የተወሰነ ነው፣ እና ከፕሮቶኮል ውጭ ይኖራል።
- ማንኛውንም ነጠላ ክስ ቆይተው መመለስ ይፈልጉ ይሆናል። ማሸግ ቋሚ ነው፣ አዲስ የተሸገ ማስተካከያ ለማሻሻል ብቸኛው ታማኝ መንገድ ነው።
- በማስገደድ ወይም በጊዜ ጫና ስር ይህን እያደረጉ ነው። ፕሮቶኮሉ ማስገደድን ያጎላል። የመከልከያ በር (myriam-kit `SPEC §8`) ለታታሪ አታሚዎችም ሆነ ለጥቆማ አቅራቢዎች ይተገበራል።

## ለእውነተኛ ማስረጃ ፋይሎች ሐሽ ማስላት

ሰነድ ፋይል በSHA-256 ለማሸግ:

```bash
shasum -a 256 my_document.pdf
```

የመጀመሪያዎቹ 64 ሄክስ ቁምፊዎች SHA-256 ናቸው። ያንን በማኒፌስት ውስጥ ያስቀምጡ።

```bash
shasum -a 256 *.pdf *.eml *.jpg > MANIFEST.txt
```

## የስራ ደህንነት ማስታወሻዎች

MYRIAM ፕሮቶኮል የስራ ደህንነት አያቀርብም። *የማስረጃ ዘላቂነት* ያቀርባል። አሁንም ይፈልጋሉ:

- ለጠበቃዎ ደህንነቱ የተጠበቀ የመገናኛ ቻናል
  ([Signal](https://signal.org/)፣ ፊት-ለ-ፊት፣ ወይም የጠበቃዎ የተቋቋመ ደህንነቱ የተጠበቀ ቻናል)
- ለመሠረታዊ ሰነዶች ደህንነቱ የተጠበቀ የማከማቻ ቦታ (በእረፍት ጊዜ የተመሰጠረ፣ ቀጣሪዎ ሊደርስ በሚችል የክላውድ አገልግሎቶች ላይ አይደለም)
- ለማሸግ ንፁህ መሣሪያ (ከቀጣሪ መለያዎች ጋር ያልተገናኘ የግል ላፕቶፕ፣ በተለይም ለዚህ ዓላማ ብቻ የተጠቀመ)
- ለማተም ንፁህ አውታረ መረብ (የቀጣሪዎ አውታረ መረብ አይደለም፣ ቀጣሪዎ ISPዎን ሊጠራ ከቻለ የቤትዎ IP አይደለም)

ለጥቆማ አቅራቢዎች የስራ ደህንነት የሚመከሩ ንባቦች:

- [The Intercept's SecureDrop documentation](https://securedrop.org/)
- [Freedom of the Press Foundation's resources](https://freedom.press/)
- [EFF's surveillance self-defense guide](https://ssd.eff.org/)

የማሸግ ሂደቱን ከመጀመርዎ በፊት ያማክሯቸው። ፕሮቶኮሉ ከመጠራቱ በፊት ስለተደረጉ መጥፎ የስራ ደህንነት ምርጫዎች አይከላከልም።

## ተቀባዮች ሊያውቁ የሚገባቸው

ለተቀባዮች ይህን README ከታሸገ ማኒፌስት ጋር ይላኩ። ማኒፌስቱ ራሱ የተቀባይ ፕሮቶኮል (FACT 06) ይዟል፣ ግን READMEው የተቀባይ ሚናን ያስፋፋል:

- **ጋዜጠኞች**: በመጀመሪያ የማኒፌስቱን የራሱ ታማኝነት ያረጋግጡ። ከዚያ ሰነዶቹን በተለየ ደህንነቱ የተጠበቀ ቻናል በኩል ይጠይቁ። ከመታተም በፊት የእያንዳንዱ ሰነድ SHA-256 ከማኒፌስት ጋር መዛመዱን ያረጋግጡ። ያለ ጥቆማ አቅራቢው ፈቃድ ሰነዶቹን አያትሙ፣ የታሸገ ማኒፌስት መኖሩንና የማረጋገጫ መደምደሚያዎን ግን ያትሙ።
- **ተቆጣጣሪዎች**: ተመሳሳይ የማረጋገጫ ሂደት። የታሸገ ማኒፌስት ከተዛመዱ ሰነዶች ጋር ከመደበኛ የጥበቃ-ሰንሰለት ስብስብ ውጭ ያለው ጠንካራው የማስረጃ ዓይነት ነው።
- **ጠበቆች**: የታሸገ ማኒፌስት ሰነዶቹ በተባለው ጊዜ ለመኖራቸው የፎረንሲክ መልሕቅ ይሰጣል። የተወሰነ የስልጣን ክልል የማስረጃ-ደንቦችን ለማሟላት የእርስዎ ግዴታ ነው።

## ይህ ኪት ምን አይደለም

- ለሕግ አማካሪ ምትክ አይደለም።
- ማንኛውም ሰው በይፋዎ እርምጃ እንደሚወስድ ዋስትና አይደለም።
- ከአጸፋ ጥበቃ አይደለም።
- የሰነዶቹ ይዘት ማረጋገጫ አይደለም (በማሸግ ጊዜ ያሉ ባይቶቻቸው ብቻ)።
- ከማንም ጋር በስም ለመግባባት መንገድ አይደለም — ማሸግ ሕዝባዊ ነው፤ የእርስዎ signing-principal ስም እውነተኛ ከሆነ፣ ተለይተዋል።

## መዝጊያ

የታሸገ ማኒፌስት ትንሽ ነገር ነው። አንድ ትልቅ የባይት ዥረቶች ስብስብ በተወሰነ ቅርጽ በተወሰነ ጊዜ ላይ መኖሩን የሚያረጋግጥ አንድ የባይት ዥረት ነው። ያ ትንሽነት ጥንካሬው ነው — አንድ ነገር ብቻ ያደርጋል፣ እና ያንን አንድ ነገር በ Bitcoin blockchain የሕይወት ዘመን ሁሉ ያደርጋል።

ይህን የሚያነቡት የጥቆማ አቅራቢ ለመሆን እያሰቡ ስለሆነ ከሆነ፣ እባክዎ: በመጀመሪያ ጠበቃ ያነጋግሩ። ከላይ ያሉትን የስራ ደህንነት መመሪያዎች ይጠቀሙ። ከዚያ፣ እና ሁሉም እነዚያ የላይኛው ጉዳዮች በሥርዓት ካሉ ብቻ፣ ይህን ኪት እንደ ይፋ ማድረግዎ የምስጠራ አካል ይቁጠሩ። ምስጠራው ቀላሉ ክፍል ነው። ከምስጠራ በላይ ያለው ሁሉ እውነተኛው አደጋ የሚኖርበት ነው።
