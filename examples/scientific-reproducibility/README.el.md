# Kit Επιστημονικής Αναπαραγωγιμότητας

Ένα έτοιμο προς χρήση παράδειγμα myriam-kit για ερευνητικές ομάδες, επιστημονικά
περιοδικά και θεσμούς εστιασμένους στην αναπαραγωγιμότητα, που επιθυμούν να δημοσιεύσουν
ένα αγκυρωμένο σε Bitcoin, με ανίχνευση παραποίησης, εκδήλωμα (manifest) αναπαραγωγιμότητας
παράλληλα με μια εργασία ή ένα προδημοσίευμα (preprint).

## Τι παράγει αυτό το kit

Μια σφραγισμένη δημοσίευση που περιέχει, για μία μελέτη:

1. Ταυτοποίηση της μελέτης (τίτλος εργασίας, συγγραφείς, ίδρυμα, χρηματοδότες, COI)
2. **SHA-256 του αρχείου ακατέργαστων δεδομένων** — το κανονικό σύνολο δεδομένων
3. **Hash της διαδικασίας ανάλυσης** — script + digest της εικόνας Docker +
   καθηλωμένες (pinned) εξαρτήσεις
4. **Hash του πακέτου αναμενόμενης εξόδου** — αυτό που θα έπρεπε να παράγει
   ένας πιστός αναπαραγωγέας
5. Πρωτόκολλο αναπαραγωγής + δεσμεύσεις δημοσίευσης μετά τα αποτελέσματα
6. Υπογράφοντα(ες) εντολέα(εις)

Η σφραγίδα προορίζεται για το στάδιο **προ-ανάλυσης**: δεδομένα παγωμένα,
διαδικασία οριστικοποιημένη, αναμενόμενη έξοδος υπολογισμένη, ΟΛΑ πριν δουν
οποιαδήποτε αποτελέσματα με σκοπό τη συγγραφή της εργασίας.

## Έναντι ποιων κινδύνων προστατεύει αυτό το kit

**Τρόποι αστοχίας που πιάνει η σφραγίδα:**

- **p-hacking**: εκ των υστέρων αλλαγές στην ανάλυση αφού φανούν τα αποτελέσματα είναι
  ανιχνεύσιμες μέσω diff μεταξύ του σφραγισμένου FACT 04 και της δημοσιευμένης διαδικασίας.
- **HARKing** (Hypothesizing After Results are Known): η
  προεγγεγραμμένη υπόθεση στο FACT 02 είναι byte-σταθερή.
- **Παραποίηση δεδομένων μετά την εργασία**: ο επανυπολογισμός του hash του αρχείου δεδομένων έναντι
  του FACT 03 αποκαλύπτει οποιαδήποτε τροποποίηση.
- **Drift διαδικασίας**: ο επανυπολογισμός του hash της διαδικασίας ανάλυσης έναντι του FACT 04
  πιάνει σιωπηρές αλλαγές.
- **Χειραγώγηση συγγραφικής ιδιότητας**: η σιωπηρή προσθήκη ή αφαίρεση συγγραφέων μετά
  τη δημοσίευση γίνεται ανιχνεύσιμη έναντι του σφραγισμένου FACT 02.

**Τρόποι αστοχίας που ΔΕΝ καλύπτονται:**

- Απάτη στη συλλογή ακατέργαστων δεδομένων (η σφραγίδα αγκυρώνει τα δεδομένα στο σημείο
  της σφράγισης, όχι της συλλογής)
- Βιολογική / εμπειρική μη αναπαραγωγιμότητα (η σφραγίδα αγκυρώνει
  μόνο την υπολογιστική αναπαραγωγιμότητα)
- Ζητήματα μεγέθους δείγματος ή εγκυρότητας πληθυσμού (αυτά είναι ζητήματα
  σχεδιασμού της μελέτης)
- Η αλήθεια του υποκείμενου ισχυρισμού (η σφραγίδα αγκυρώνει τι ισχυρίστηκαν,
  όχι αν ο κόσμος αντιστοιχεί στον ισχυρισμό)

## Πότε να χρησιμοποιήσετε αυτό το kit

Σκεφτείτε τη σφράγιση αν:

- Είστε ερευνητική ομάδα που εκτελεί μια ανάλυση υψηλού διακυβεύματος (κλινική
  δοκιμή, αξιολόγηση συστήματος ΤΝ, μελέτη επαναληψιμότητας) και θέλετε δικανική
  ακεραιότητα πέρα από τα χρονικά σήματα του OSF / GitHub
- Είστε ερευνητής/τρια σε πρώιμο στάδιο σταδιοδρομίας και θέλετε ασύμμετρη φήμη
  ως προστασία: η σφραγισμένη προ-ανάλυση σημαίνει ότι μελλοντική κριτική «έκαναν p-hacking»
  έχει δημόσιο διαψεύστη
- Είστε εκδότης περιοδικού ή επικεφαλής έργου επαναληψιμότητας και θέλετε να
  ενθαρρύνετε προεγγραφή κρυπτογραφικής ποιότητας για ένα υποσύνολο
  εργασιών
- Εργάζεστε σε μια ανάλυση όπου οι εκ των υστέρων τροποποιήσεις έχουν ουσιαστικές
  συνέπειες στο πεδίο (ψυχολογία, συμπεριφορική οικονομική, ιατρική έρευνα)

## Ιστορικό προηγούμενο θνησιμότητας / βλάβης

Η συνάφεια θνησιμότητας αυτού του kit είναι έμμεση:

- **Μη αναπαραγωγιμότητα στην έρευνα βλαστοκυττάρων / καρκίνου:** αρκετές εργασίες
  έχουν οδηγήσει σε βλάβη ασθενών μέσω παραπλανημένων κλινικών δοκιμών. Η σφραγισμένη
  προεγγραφή της αρχικής διαδικασίας θα είχε αγκυρώσει τη
  σύγκριση.
- **Διαμάχη αποτελεσματικότητας του Tamiflu:** η ανασκόπηση Cochrane χρειάστηκε
  χρόνια εν μέρει επειδή η πρόσβαση σε ακατέργαστα δεδομένα ήταν αμφισβητούμενη. Η σφράγιση
  των hashes δεδομένων κατά τη δημοσίευση θα είχε καταστήσει τις διαφορές πρόσβασης σε δεδομένα
  άμεσα επιλύσιμες.
- **Γενική κρίση μη αναπαραγωγιμότητας:** το κόστος θνησιμότητας σε επίπεδο πεδίου
  της μη αναπαραγώγιμης βιοϊατρικής έρευνας είναι σημαντικό (οι Begley &
  Ellis 2012 εκτίμησαν >50% των προκλινικών ευρημάτων ως μη αναπαραγώγιμα).
  Η κρυπτογραφική αγκύρωση είναι ένας δομικός μοχλός.

## Πώς να κάνετε fork αυτό το kit

```bash
cp -r ~/Genesis/myriam-kit/examples/scientific-reproducibility ~/my-paper-seal
cd ~/my-paper-seal

# CRITICAL: do this BEFORE seeing any results
shasum -a 256 raw_data_v0.tar.gz       # compute the canonical data hash
shasum -a 256 analysis.py              # hash your pipeline
docker buildx imagetools inspect <image> --format '{{.Manifest.Digest}}'

$EDITOR facts/*.txt                    # populate with real values
bash build/seal.sh                     # seal BEFORE running the analysis
python3 build/verify.py                # confirm

# Now run the analysis. Hash the output and confirm matches FACT 05.
# If it doesn't match, the pipeline is non-deterministic; fix and re-seal.
```

## Ενσωμάτωση με υφιστάμενη ακαδημαϊκή υποδομή

- **arXiv / bioRxiv / medRxiv**: συμπεριλάβετε το URL της σφραγίδας MYRIAM ή το IPFS CID
  στην περίληψη ή στο συμπληρωματικό υλικό
- **OSF**: επισυνάψτε το τεχνούργημα σφραγίδας στο έργο OSF παράλληλα με τη
  συμβατική φόρμα προεγγραφής
- **AsPredicted**: συμπεριλάβετε το URL της σφραγίδας MYRIAM στο έγγραφο
  προεγγραφής
- **GitHub**: το hash της δέσμευσης (commit) της διαδικασίας ανάλυσης είναι αυτό που αγκυρώνει
  το FACT 04· το GitHub παρέχει την πηγή, το MYRIAM αγκυρώνει τα bytes
- **Zenodo / Figshare / OpenAIRE**: καταθέστε τα ακατέργαστα δεδομένα και στη συνέχεια σφραγίστε
  το hash της κατάθεσης· η σφραγίδα είναι δικανικά πιο δύσκολο να αντικατασταθεί σιωπηρά
  από τα μεταδεδομένα της κατάθεσης
- **Συμπληρωματικό υλικό περιοδικού**: συμπεριλάβετε το τεχνούργημα σφραγίδας MYRIAM
  ως συμπληρωματικό αρχείο κατά την υποβολή

## Τι ΔΕΝ αντικαθιστά αυτό το kit

- Έγκριση IRB / επιτροπής δεοντολογίας
- Αξιολόγηση από ομοτίμους (peer review)
- Μελέτες επαναληψιμότητας (η υπολογιστική αναπαραγωγιμότητα ≠ εμπειρική
  αναπαραγωγιμότητα)
- Συμφωνίες πρόσβασης σε δεδομένα για ευαίσθητα δεδομένα

## Σημείωση για τη σειρά συγγραφέων και την απόδοση πιστώσεων

Η σφραγίδα αγκυρώνει τη ΛΙΣΤΑ ΣΥΓΓΡΑΦΕΩΝ κατά τη στιγμή της σφράγισης. Διαφωνίες σχετικά με
τη συγγραφική ιδιότητα που προκύπτουν μετά τη δημοσίευση μπορούν να αξιολογηθούν έναντι της
σφραγισμένης λίστας. Πρόκειται για έναν μικρό μοχλό υπευθυνότητας — σημαντικό για ερευνητές/τριες
σε πρώιμο στάδιο σταδιοδρομίας που προστίθενται ή αφαιρούνται σιωπηρά.
