  2. Pulls the Docker image matching FACT 04's digest

above. If the reproducer's output hash does not match, one of the
following is true:

  - The reproducer received non-canonical raw data (re-verify FACT 03)
  - The reproducer ran a different pipeline (re-verify FACT 04)
  - The pipeline is genuinely non-deterministic (a bug; the
    pre-registered pipeline should be fully deterministic)

    that overrides the original; sealed result is now contestable)

This fact asserts only the canonical output hash for the fictional
analysis. It does not assert that the result is biologically /
clinically / sociologically meaningful, only that it is the bytewise
output of running the sealed pipeline on the sealed data.

THE DISTINCTION matters: the seal anchors COMPUTATIONAL reproducibility.
It does not anchor BIOLOGICAL / EMPIRICAL reproducibility (whether the
finding would hold in a different sample or under different conditions).
