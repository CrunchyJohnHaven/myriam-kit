## អ្វីដែលឧបករណ៍នេះការពារ

ករណីប្រវត្តិសាស្ត្រនៃអត្រាមរណភាពដែលឧបករណ៍នេះអាចជួយបាន៖

- **Vioxx (Merck, 2004)**: ព្រឹត្តិការណ៍សរសៃឈាមបេះដូងគឺជា endpoint
  ដែលបានចុះឈ្មោះមុនក្នុងការសាកល្បង APPROVe ប៉ុន្តែការវិភាគដែលបាន
  បោះពុម្ពប្រើ cutoff ដែលបន្ថយការបង្ហាញគ្រោះថ្នាក់។ ត្រានៃ cutoff
  ដែលបានចុះឈ្មោះមុននឹងធ្វើឱ្យការផ្លាស់ប្តូរ cutoff ក្រោយការសាកល្បង
  អាចរកឃើញដោយសាធារណៈមុនពេលក្រដាសបោះពុម្ព។
- **Paxil/Study 329 (GSK/SmithKline, 2001)**: ទិន្នន័យអវិជ្ជមាននៃការ
  សាកល្បងលើគំនិតធ្វើបាបខ្លួនឯងរបស់យុវវ័យត្រូវបានដាក់ស៊ុមថ្មីតាមការ
  វិភាគក្រោយការសាកល្បង។ ការចុះឈ្មោះមុនដែលបានបិទត្រានៃ endpoint នឹង
  ធ្វើឱ្យការដាក់ស៊ុមថ្មីអាចរកឃើញដោយសាធារណៈ។
- **ការសាកល្បងជំងឺមហារីកច្រើន** ដែល endpoint នៃការរស់រានត្រូវបានប្តូរ
  ទៅ endpoint surrogate (progression-free survival, response rate)
  នៅពេលដែលអត្ថប្រយោជន៍នៃការរស់រានមិនកើតឡើង។

**របៀបនៃការបរាជ័យដែលត្រាចាប់បានដោយផ្ទាល់៖**

- **ការប្តូរ endpoint**: ការដំឡើង endpoint ទីពីរក្រោយការសាកល្បងទៅ
  "primary" នៅពេលដែល primary ពិតប្រាកដបរាជ័យ។ FACT 05 នៃត្រាមាន
  primary endpoint ដែលបានចុះឈ្មោះមុន; primary endpoint ដែលបោះពុម្ព
  ត្រូវតែផ្គូផ្គង។
- **ការជ្រើសរើស outcome**: ការរាយការណ៍ដោយជ្រើសរើសតែការវិភាគដែលធ្វើ
  ការ។ FACT 06 មានផែនការវិភាគពេញលេញដែលបានចុះឈ្មោះមុន; ការវិភាគដែល
  បោះពុម្ពត្រូវរួមបញ្ចូលអ្វីដែលបានចុះឈ្មោះមុនទាំងអស់។
- **ការរុករក subgroup ក្រោយការសាកល្បង**: ការអះអាងថា "ឱសថដំណើរការ
  សម្រាប់ subgroup X" ក្រោយការមើលទិន្នន័យ។ FACT 07 មានបញ្ជី subgroup

  អន្តរការ។ FACT 06 មានទំហំសំណាក និងគណនាអំណាចដែលបានចុះឈ្មោះមុន។
- **ការដកស្ងាត់ៗនៃ endpoint ទីពីរដែលមិនស្រួល**: FACT 05 ដាក់លំដាប់
  endpoint ទីពីរសម្រាប់ការសាកល្បងតាមលំដាប់; ការដាក់លំដាប់ឡើងវិញ

- **ការទប់ស្កាត់លទ្ធផលអវិជ្ជមាន**: FACT 08 មានការតាំងសច្ចានៃការ
  បោះពុម្ព; ការបរាជ័យក្នុងការបោះពុម្ពក្នុងបង្អួចដែលបានកំណត់
