## ពេលណាគួរប្រើឧបករណ៍នេះ

  កំណត់ត្រាការបញ្ចេញលើសពីអ្វីដែលអ្នកគ្រប់គ្រងតម្រូវឱ្យ

  មិនពាណិជ្ជកម្ម (ឧ. សម្រាប់ជំងឺដែលត្រូវបានគេមិនអើពើ)
