# தடுப்பூசி தொகுதி சான்றளிப்பு கிட்

*தமிழ் மொழிபெயர்ப்பு — அசல் ஆங்கிலப் பதிப்பிற்கு, பார்க்கவும் [README.md](README.md)*

தடுப்பூசி உற்பத்தியாளர்கள், தர உறுதிப்பாட்டு அதிகாரிகள், மற்றும் பொது
சுகாதார நிறுவனங்களுக்கான ஒரு உடனடி பயன்பாட்டுக்கு தயாரான myriam-kit
எடுத்துக்காட்டு, இவர்கள் ஒரு தொகுதி வெளியீட்டின் Bitcoin-நங்கூரமிடப்பட்ட,
சேதம்-வெளிப்படையான முத்திரையை வெளியிட விரும்புகின்றனர்.

## இந்த கிட் எதற்கு எதிராக பாதுகாக்கிறது

**முத்திரை பிடிக்கும் தோல்வி முறைகள்:**

- **வெளியீட்டுச் சோதனை முடிவுகளின் அமைதியான பின்னோக்கி திருத்தம்.**
  எடுத்துக்காட்டில் உள்ள அனைத்து 8 வெளியீட்டுச் சோதனை PDF-களும்
  ஹாஷ் செய்யப்படுகின்றன. வெளியீட்டிற்குப் பிறகு மீண்டும் ஹாஷ் செய்வதன்
  மூலம் எந்த திருத்தமும் வெளிப்படுகிறது.
- **வெளியீட்டின் போது எந்த விவரக்குறிப்புகள் பொருந்தும் என்பது குறித்த
  தகராறுகள்.** முத்திரையிடப்பட்ட FACT 02 இல் நியதி தொகுதி அடையாளம்
  மற்றும் உற்பத்தி-பதிவு ஹாஷ் உள்ளது. எதிர்காலத்தில் "விவரக்குறிப்பு
  எப்போதும் X ஆகவே இருந்தது" என்ற தகராறுகளுக்கு ஒரு பொது நங்கூரம்
  இருக்கிறது.
- **அமைதியான திரும்பப்பெறுதல்-நோக்கம் குறைப்பு.** FACT 05 இல்
  மருந்துக் கண்காணிப்பு (pharmacovigilance) உறுதிமொழிகள் முத்திரையிடப்பட்டுள்ளன;
  அவற்றை மதிக்கத் தவறுவது பொதுவாக கண்காணிக்கக்கூடியது.
- **விவரக்குறிப்புக்கு வெளியே உள்ள தொகுதிக்கான பின்னோக்கி மறு-விவரக்குறிப்பு.**
  ஒரு வெளியீட்டுச் சோதனை முடிவு உண்மையில் எல்லைக் கோட்டு PASS ஆக
  இருந்து, வெளியீட்டிற்குப் பிறகு FAIL என மறுவகைப்படுத்தப்பட்டால்,
  முத்திரையிடப்பட்ட அறிக்கை ஹாஷ்கள் அசல் முடிவு PDF என்ன சொன்னது
  என்பதை நிரூபிக்கின்றன.

**உள்ளடக்கப்படாத தோல்வி முறைகள்:**

- உற்பத்தி செயல்முறையிலேயே மோசடி (முத்திரை சோதனை முடிவுகளை நங்கூரமிடுகிறது,
  உற்பத்தி ஒருமைப்பாட்டை அல்ல).
- மற்றபடி தேர்ச்சி பெற்ற தொகுதியின் உயிரியல் விளைவாக ஏற்படும்
  பாதகமான நிகழ்வுகள் (முத்திரை மருத்துவ பாதுகாப்பை சான்றளிக்கவில்லை,
  ஆவணப்பதிவு ஒருமைப்பாட்டை மட்டுமே).
- எந்த ஒழுங்குமுறை விவரக்குறிப்பு பதிப்பு அதிகாரபூர்வமானது என்பது
  குறித்த தகராறுகள்.

## இந்த கிட்டை எப்போது பயன்படுத்த வேண்டும்

பின்வரும் சூழ்நிலைகளில் ஒரு தொகுதி வெளியீட்டை முத்திரையிட பரிசீலிக்கவும்:

- நீங்கள் ஒரு தடுப்பூசி உற்பத்தியாளராக இருந்து, ஒழுங்குமுறை அமைப்புகள்
  தேவைப்படுவதைவிட அதிகமான வெளியீட்டுப் பதிவுகளின் தடயவியல் ஒருமைப்பாட்டை
  நிரூபிக்க விரும்பினால்
- நீங்கள் "நாங்கள் என்ன சான்றளிக்கிறோமோ அதை வெளியிடுவோம்" என்ற
  நடைமுறையை ஏற்றுக்கொள்ளும் ஒரு QA குழுவாக இருந்தால்
- நீங்கள் பலவீனமான ஒழுங்குமுறை உள்கட்டமைப்பு கொண்ட ஒரு அதிகார
  வரம்பில் ஒரு சிறிய/தேசிய-அளவிலான உற்பத்தியாளராக இருந்து, ஒரு
  உலகளாவிய சரிபார்ப்பாளருக்கு எதிராக தொகுதிப் பதிவுகளை நங்கூரமிட
  விரும்பினால்
- நீங்கள் ஒரு பொது-சுகாதார நிறுவனமாக அல்லது NGO ஆக இருந்து, ஒரு
  இலாப-நோக்கற்ற தடுப்பூசி உற்பத்தித் திட்டத்தை (எ.கா. புறக்கணிக்கப்பட்ட
  நோய்களுக்கு) நடத்துகிறீர்களென்றால்

## வரலாற்று இறப்பு முன்னுதாரணம்

- **ஹெபரின் மாசுபாடு (Heparin 2008, சீனா):** ஹெபரின் தொகுதிகளில் கள்ள
  ஓவர்சல்பேட்டட் காண்ட்ராய்டின் சல்பேட். முத்திரையிடப்பட்ட
  வெளியீட்டுச் சோதனை மேனிஃபெஸ்ட் சோதனைப் பதிவின் அமைதியான
  வெளியீட்டிற்குப் பிந்தைய திருத்தத்தை குறியாக்கவியல் ரீதியாக
  கண்டறியக்கூடியதாக ஆக்கியிருக்கும்.
- **டைலினால் சயனைடு (Tylenol 1982):** சேதத்தின் தோற்றுவாய் குறித்த
  பின்னோக்கி தகராறுகள். ஒரு முத்திரையிடப்பட்ட உற்பத்தி-பதிவு ஹாஷ்
  வழங்கலுக்கு முந்தைய நிலையை நங்கூரமிடுகிறது.
- **பல்வேறு தடுப்பூசி தொகுதி திரும்பப்பெறுதல்கள்** இவற்றில் "இந்த
  தொகுதி எப்போதும் முறையாக சோதிக்கப்பட்டதா" என்ற கேள்வி பல
  ஆண்டுகளுக்குப் பிறகு சர்ச்சைக்குரியதாக மாறுகிறது.

இந்த கிட் கடுமையான அர்த்தத்தில் இவற்றில் எதற்கும் ஒரு தீர்வு அல்ல —
ஆனால் ஒவ்வொரு வழக்கிலும், முத்திரையிடப்பட்ட தொகுதி சான்றளிப்புகள்
ஒரு தெளிவான தடயவியல் காலவரிசையை வழங்கியிருக்கும்.

## உண்மையான தொகுதிக்காக இந்த கிட்டை எவ்வாறு ஃபோர்க் செய்வது

```bash
cp -r ~/Genesis/myriam-kit/examples/vaccine-batch-attestation ~/release-FBP-XXXX
cd ~/release-FBP-XXXX
$EDITOR facts/*.txt       # replace fictional content with real batch
bash build/seal.sh        # SEAL BEFORE distribution begins
python3 build/verify.py   # confirm
```

**முக்கியமான நேர-நிர்ணயம்**: QA வெளியீட்டின் தருணத்தில் முத்திரையிடவும்,
தொகுதி வழங்கப்படுவதற்கு **முன்பு**. வழங்கலுக்குப் பிந்தைய முத்திரை
தடயவியல் ரீதியாக பலவீனமானது.

## தற்போதைய மருந்து உள்கட்டமைப்புடன் ஒருங்கிணைப்பு

- **eCTD சமர்ப்பிப்புகள் (FDA / EMA):** முத்திரையிடப்பட்ட வெளியீட்டு
  URL அல்லது IPFS CID Module 3 (தரம்) பிரிவில் குறிப்பிடப்படலாம்
- **WHO Prequalification:** முத்திரையிடப்பட்ட மேனிஃபெஸ்ட் FDA/EMA
  அல்லாத அதிகார வரம்புகளுக்கு வெளியீட்டிற்கு முன்னான சோதனைச்
  சங்கிலியை நிரூபிக்கிறது
- **மருந்துக் கண்காணிப்பு (Pharmacovigilance) பின்னூட்டம்:** FACT 05
  உற்பத்தியாளரை 6 மற்றும் 12 மாதங்களில் மொத்த AE எண்ணிக்கையின்
  முத்திரையிடப்பட்ட அறிக்கைக்கு உறுதியளிக்கிறது
- **திரும்பப்பெறுதல் அமைப்புகள் (RxConnect, MedWatch):** திரும்பப்பெறுதல்
  தேவைப்பட்டால், திரும்பப்பெறுதல் நோக்கமே MYRIAM-இன் கீழ் முத்திரையிடப்படும்,
  அனைத்து வழங்கப்பட்ட தொகுதிகளும் கையாளப்படுகின்றன என்பதற்கு பொது
  சரிபார்ப்பை வழங்குகிறது

## இந்த கிட் எதை மாற்றவில்லை

- FDA / EMA / WHO ஒழுங்குமுறை சமர்ப்பிப்புகள்
- GMP தணிக்கை தடயங்கள் (முத்திரை முடிவுகளை நங்கூரமிடுகிறது, மூல
  செயல்முறை பதிவுகளை அல்ல)
- மருந்துக் கண்காணிப்பு தரவுத்தளங்கள் (VAERS / EudraVigilance)
- திரும்பப்பெறுதல் ஒருங்கிணைப்பு அமைப்புகள்
