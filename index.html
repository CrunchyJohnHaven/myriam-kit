myriam-kit

CC0 v0.2.0 6 example kits

A small CC0 toolkit for publishing a small set of facts such that:

  • Any change to any fact is publicly detectable (Merkle tree)
  • The state at publication is verifiable against the Bitcoin blockchain (OpenTimestamps)
  • The content survives takedown of any single hosting provider (IPFS)
  • The whole thing weighs under 100 KB and is verifiable with ~50 lines of Python

Not a blockchain. No token. No DAO. No consensus mechanism to participate in. The tool is sha256sum + ots + ipfs, glued together with good naming conventions.

Get started

git clone https://github.com/CrunchyJohnHaven/myriam-kit
cd myriam-kit
bash lib/myriam-init.sh my-publication
cd my-publication
$EDITOR facts/*.txt
bash ../lib/seal.sh

See QUICKSTART.md for the full 5-minute walkthrough.
