## මෙය නොකරන දේ

  ප්‍රකාශනය සඳහා වන අතර, සැඟවීම සඳහා නොවේ.
- **විශාල දත්ත කට්ටල කරා පරිමාණය වැඩිකිරීම.** Merkle ගස හොඳින් පරිමාණය වේ; පාඨකයෙකුගේ
  අවධානය එසේ නොකරයි. කරුණු ~20කට පසුව පිටුව කියවිය නොහැකි වේ. ඔබට
  වාර්තා මිලියනයක් මුද්‍රා කිරීමට අවශ්‍ය නම්, ඒවා පිටුවෙන් පිට හැෂ් කර
  එම පැති-ගසේ Merkle මූලය එක් කරුණකට දමන්න.
- **අහෝසි කිරීමට එරෙහිව ආරක්ෂා වීම.** මුළු කාරණයම නම් ඔබ මුද්‍රා කළ දේ වෙනස් කළ නොහැකි බවයි.
  ඔබට පසුව ප්‍රකාශයක් අහෝසි කිරීමට අවශ්‍ය විය හැකි නම්, එය මුද්‍රා නොකරන්න.
